## Jnana Management System

Jnana Management System is a syllabus management dashboard for exams and their hierarchy:
Exam → Sub Exams → Syllabus → Chapters → Sections.

## Features

- Single admin login
- CRUD for exams, syllabus, chapters, and sections
- Dashboard overview with counts

## Local development

- Install dependencies: `npm install`
- Start the dev server: `npm run dev`
